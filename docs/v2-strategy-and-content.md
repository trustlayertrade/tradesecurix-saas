# TradeSecurix V2 Website Blueprint

## 1) Brand strategy
- **Positioning:** TradeSecurix is a B2B trade risk workflow platform that helps teams verify counterparties, review trade documents, and make auditable approval decisions.
- **Promise:** Better trust decisions before money or goods move.
- **Tone:** Calm, precise, institutional-grade, partnership-ready.
- **Differentiation:** Not a generic dashboard. A focused approval and auditability workflow for practical trade operations.

## 2) Sitemap
- Home (`index.html`)
- Product (`product.html`)
- Solutions (`solutions.html`)
- Partnerships (`partnerships.html`)
- Security (`security.html`)
- Pricing (`pricing.html`)
- About (`about.html`)
- Contact / Book Demo (`contact.html`)
- FAQ (`faq.html`)
- Privacy Policy (`privacy.html`)
- Terms (`terms.html`)
- Cookie Policy (`cookies.html`)
- Thank You (`thank-you.html`)

## 3) Homepage wireframe
1. Header/navigation + dual CTA
2. Hero with single core message + polished product visual placeholder
3. Audience fit cards

5. Four-step process
6. Core capability cards
7. Partnerships section
8. Security/trust section
9. Pricing preview
10. FAQ preview

12. Footer with legal links

## 4) Homepage copy
Implemented directly in `index.html` with premium, concise conversion language and placeholders where proof is unavailable.

## 5) Product page wireframe + copy
- Hero, product workflow blocks, module summary, implementation approach, CTA.

## 6) Solutions page wireframe + copy
- Hero + audience-specific value blocks covering all target segments.

## 7) Partnerships page wireframe + copy
- Dedicated partnership page for chambers, associations, export support organizations, consultants, and referral partners.

## 8) Security page wireframe + copy
- Access control, audit logging, governance alignment, plus compliance placeholder policy.

## 9) Pricing page wireframe + copy
- Three clear tiers with quote placeholders (no fake pricing claims).

## 10) About page wireframe + copy
- Mission, principles, and operating philosophy.

## 11) Contact page wireframe + copy
- Separate forms for Book Demo and Request Sample Review.

## 12) FAQ copy
- Six practical FAQs focused on trust, scope, and adoption.

## 13) Footer structure
- Brand summary
- Main page links
- Legal links

## 14) Design system
- **Colors:** navy trust tones, muted slate neutrals, white surfaces, subtle line separators.
- **Typography:** Inter/Segoe/Roboto stack for clean enterprise readability.
- **Spacing:** generous vertical rhythm, low-clutter card system.
- **Visual principles:** minimal polished UI, no fake data overload.

## 15) Component system
- Sticky header + nav
- CTA buttons (primary/secondary)
- Card grid (2/3/4 columns)
- Hero with visual container
- Band sections for contrast
- Footer columns
- Form inputs and textareas

## 16) SEO titles and descriptions
Defined per page in `<title>` and `<meta name="description">` blocks across all HTML pages.

## 17) Final V2 concept
A premium, short, and serious B2B web presence focused on credibility, clarity, and conversion to demo/sample/partnership leads.

## 18) Production-ready implementation
Delivered as a complete static multi-page website with shared design system assets:
- `assets/styles.css`
- `assets/main.js`
- All required site pages listed in sitemap
